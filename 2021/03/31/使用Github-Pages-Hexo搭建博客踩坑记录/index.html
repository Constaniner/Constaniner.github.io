使用Github Pages + Hexo搭建博客踩坑记录

  1. Hexo d 无法部署到Github

    检查站点配置文件（不是主题配置文件）的deploy部分

  2. 使用的主题报错 / 预览跟Demo差很多 / 无法使用

    建议新手使用诸如Next、Fluid等有较多人维护，仍然活跃的主题，这类主题通常有完善的配置引导供你入门。

    如果看英文有困难的话，这里贴两个Next主题中文指引、Next文档，别的主题找找也会有类似的文章。

  3. 站点配置文件怎么写 / 各个关键字都是什么意思

    请参考Hexo官方文档，写的很清楚Hexo

  4. Hexo g报错failed to detect version info, cannot read property 'replace' of undefined

    大概率是Github连接问题。将你的梯子调整成全局 / pac模式能解决

  1. Mac 用户安装 Node.js 报 EACESS 权限错误

    请参考npm官方文档，强烈建议不要使用root覆盖安装

  2. 找不到好看的主题

    这里推荐几个，主要是简约 / Material Design风格的

    Next

    Fluid

    Ayer

    Material

    ICARUS

  3. 找不到好看的icon

    Font Awesome
